## 菜品特色说明

### 标签含义
- **招牌**: 本店特色菜品,强烈推荐
- **必点**: 顾客必点菜品,销量领先
- **推荐**: 厨师推荐,品质保证
- **热销**: 近期热卖,人气爆棚
- **川菜**: 正宗四川风味
- **素菜**: 适合素食者
- **健康**: 低脂健康选择
- **海鲜**: 新鲜海味
- **主食**: 填饱肚子的好选择
- **汤品**: 滋养汤羹
- **甜品**: 甜蜜收尾
- **饮品**: 清爽解腻
- **小吃**: 特色小食
- **新鲜**: 当日新鲜制作

### 价格说明
- 所有菜品都显示**原价**和**优惠价**
- 原价显示为删除线样式
- 优惠价为实际结算价格
- 部分菜品享有折扣优惠

### 库存管理
- 每道菜品都有库存数量
- 库存不足时会在界面上提示
- 建议提前预约或改选其他菜品

## 菜品数据结构

```javascript
{
  id: 菜品ID,
  categoryId: 分类ID,
  name: 菜品名称,
  price: 优惠价格,
  originalPrice: 原价,
  image: 图片路径,
  sold: 销量,
  rating: 评分(1-5),
  description: 菜品描述,
  tags: 标签数组,
  stock: 库存数量,
  hot: 是否热销(true/false)
}
```

## 使用说明

1. 在微信小程序中打开
2. 浏览各分类菜品
3. 查看菜品详情、价格、评价
4. 点击"加入购物车"
5. 在购物车中结算
6. 完成下单
